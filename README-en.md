# List of features
## Beautify
  * Static wallpaper setting
  * Dynamic wallpaper setting
  * Mouse effect
  * Desktop effect
  * Taskbar setting
  * Context menu setting
  * Boot image setting
  * other
## System Tools
  * boot management
  * service management
  * background network management
  * windows settings
  * gpedit & register
  * fast search(taskbar win+q)
## Utilities
  * hardware information
  * startup assistant
  * fast run
