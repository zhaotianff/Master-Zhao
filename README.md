# Master-Zhao
Master-Zhao是Windows的一个工具集软件。 
07年时，在一个软件盘上，我第一次接触了Windows优化大师，当时的鲁大师还是作为Windows优化大师的一个附带工具安装的。作者鲁锦，是我心中当之无愧的大佬之一。    
10多年过去了，现在我也掌握了一些编程方面的知识，于是我想实现一下童年的梦想，写一个工具类的软件集合，暂且命名为赵大师，哈哈哈哈。

# 功能列表
## 美化
  * 静态壁纸设置
  * 动态壁纸设置
  * 鼠标特效
  * 任务栏设置
  * 右键菜单管理
  * 开机图像修改
  * 其它
## 系统工具
  * 启动项管理
  * 服务管理
  * 后台网络管理
  * 系统设置
  * 组策略和注册表管理
  * 快速搜索（基于任务栏）
## 实用工具
  * 硬件信息
  * 开机助手
  * 快速启动
